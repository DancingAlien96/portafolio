# Portafolio — Cristófer

Sitio estático generado: `index.html`, `styles.css`, `script.js`.

**Características:**
- Animación de texto tipo prompt
- Avatar flotante con hover effect
- Fondo de circuitos animado (líneas y nodos con efecto glow)
- Profile card con chips de habilidades
- Diseño responsive

Vista rápida en local:

```bash
# Servir con Python 3
python -m http.server 8000

# Abrir http://localhost:8000 en el navegador
```
