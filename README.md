# Portafolio — Cristófer Pérez

Portafolio profesional con animaciones de circuitos, proyectos destacados, reconocimientos y experiencia laboral.

## 🚀 Características

- ✨ Animación de texto tipo prompt
- 🎨 Avatar flotante con hover effect
- ⚡ Fondo de circuitos animado (líneas y nodos con efecto glow)
- 💼 Profile card con chips de habilidades
- 🏆 Sección de reconocimientos y logros
- 📊 Timeline de experiencia laboral animado
- 🎯 Proyectos destacados con imágenes y detalles
- 📱 Diseño completamente responsive

## 🚀 Despliegue

### GitHub Pages (Recomendado - Gratis)

1. Ve a tu repositorio en GitHub
2. Haz clic en **Settings** > **Pages**
3. En "Source", selecciona la rama `main` y carpeta `/ (root)`
4. Guarda y espera unos minutos
5. Tu sitio estará disponible en: `https://dancingalien96.github.io/portafolio/`

### Netlify (Gratis)

1. Ve a [netlify.com](https://netlify.com)
2. Haz clic en "Add new site" > "Import an existing project"
3. Conecta con GitHub y selecciona el repo `portafolio`
4. Deja la configuración por defecto y despliega
5. Tu sitio estará disponible en: `https://tu-nombre.netlify.app`

### Vercel (Gratis)

1. Ve a [vercel.com](https://vercel.com)
2. Haz clic en "New Project"
3. Importa desde GitHub el repo `portafolio`
4. Despliega con la configuración por defecto
5. Tu sitio estará disponible en: `https://portafolio-tu-nombre.vercel.app`

## 📝 Personalización

Para personalizar el contenido:

1. **Textos del hero**: Edita las frases en `script.js` (línea 1-5)
2. **Información personal**: Actualiza el profile card en `index.html`
3. **Proyectos**: Reemplaza las imágenes y edita el contenido en la sección `#projects`
4. **Reconocimientos**: Actualiza imágenes y textos en la sección `#achievements`
5. **Experiencia**: Modifica el timeline en la sección `#experience`
6. **Colores**: Ajusta las variables CSS en `:root` en `styles.css`
